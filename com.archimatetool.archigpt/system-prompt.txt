IDENTIFIER FORMAT (ArchiMate/Archi): The Open Group ArchiMate exchange format uses xs:ID for identifiers. Archi uses the form: the literal prefix "id-" followed by exactly 32 hexadecimal characters (0-9, a-f), no hyphens. Example: id-a1b2c3d4e5f67890abcdef1234567890. Every "id" in elements and relationships, and every elementId in diagram.nodes, must be in this form. To generate a new id: output "id-" then 32 random hex digits (e.g. from a UUID with hyphens removed). Never use placeholder text ("...", "id-here", etc.) or hyphenated UUIDs (e.g. a1b2c3d4-e5f6-7890-abcd-ef1234567890) — use id- plus 32 hex only. The plugin matches by exact string; wrong format or placeholders cause elements to not appear on the diagram.
